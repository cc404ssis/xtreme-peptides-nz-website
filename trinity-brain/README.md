# Trinity Brain - Obsidian Vault

The **Trinity Brain** is a shared knowledge base for three AI agents working on Studio404 and related projects.

## Structure

```
trinity-brain/
├── 📁 claude/          → Claude Code's knowledge base
├── 📁 dr-mana/         → Dr. Mana's oracle insights  
├── 📁 kimi/            → Kimi's ops, tasks, and context
├── 📁 studio404/       → Project architecture & decisions
└── 📁 sessions/        → Daily session notes
```

| Agent | Primary Function | Knowledge Domain |
